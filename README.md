## Gemini Pro Vision Client

Chat with image using Gemini Pro Vision.  
使用 Gemini Pro Vision 实现与图片对话

### Getting Started

Set `GEMINI_API_KEY` in .env  
在 .env 里设置 `GEMINI_API_KEY`

run 执行

```
npm i
```

Then 然后

```
npm run dev
```

Open then link printed in the console.  
打开控制台打印的链接
